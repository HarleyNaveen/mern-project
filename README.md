# MERN Student CRUD Project

A full-stack CRUD application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Tech Stack

- MongoDB (Database)
- Express.js (Backend Framework)
- React.js (Frontend Library)
- Node.js (Runtime Environment)

## Project Structure

mern-project/
│
├── backend
│   ├── config
│   ├── controllers
│   ├── models
│   ├── routes
│   └── server.js
│
└── frontend (Vite + React)

## Features

- Add Student
- Edit Student
- Delete Student
- View Student List
- REST API Integration
- MongoDB Database Connection

## Installation & Setup

### Backend Setup

cd backend  
npm install  
npm start  

### Frontend Setup

cd frontend/vite-project  
npm install  
npm run dev
